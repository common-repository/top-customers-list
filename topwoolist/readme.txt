== Description ==

A simple plugin to show the largest spending customer on your website to admins and website visitors/users

== Frequently ASked Questions ==
= Why am i see weird entries? =

You may have orders with incomplete data in your woocommerce orders, this can happen if a user has been deleted with record of the user causing linking of different data to fail for example

= I have just installed woocommerce but cant see any users? =

This plugin only lists by completed orders, so if you do not have any completed orders on your site, the list will be empty.

= Where do i report bugs?=

A support page is in the midst of being set up, it will be linked here

== Installation ==
1. Make sure that woocommerce is installed and activated
2. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Use the Settings->(Top Customers List) to configure the plugin

== Features ==
* Shows a widget in admin dashboard listing the biggest spending customers on your site
* Widget to show biggest spending customers anywhere else on your site that you like
* Filter what is displayed on the list by user role, time range or latest time
* Very simple plugin, when you dont want to worry about styling
